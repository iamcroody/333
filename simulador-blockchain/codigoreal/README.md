# Blockchain Simulator

**Author:** Jose Alejandro Jimenez

A concise, interactive blockchain simulator designed to visualize core concepts like Hashing, Mining (Proof of Work), and immutable ledgers.

## Overview

This project simulates a basic blockchain network where users can:
- Create wallets and transactions.
- Mine blocks with adjustable difficulty.
- Visualize the chain validation (Hash pointers).
- Attempt to tamper with data to see chain invalidation.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the simulator:
   ```bash
   npm run dev
   ```

Built with **React**, **TypeScript**, and **Vite**.
